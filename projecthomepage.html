City Latitude vs Climate

This project analyzes how latitude impacts various aspects of climate. This was achieved with a data set comprised of 546 cities and their measured cloudiness, humidity, max temp, and wind speed for a particular date. This data was sourced using an OpenWeather API, and visualized with MatPlotLib in Python.
